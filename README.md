Блиотека для Python написанная на C#. Полноценный 2D движок, который похож на PyGame, но с большим функционалом

A Python library written in C#. A full-fledged 2D engine similar to PyGame, but with more functionality

# GetStartded RU

## alien python

`alien` - каталог Python проекта, который включает в себя все необходимые команды для работы с движком. Подробнее читайте в [документации](./dosc/ru/index.md)

Для работы создайте и активируйте виртуальное окружение

```
cd alien

# For MacOS / Linux
python3 -m venv venv
source venv/bin/activae

# For Windows
python -m venv venv
venv\Scripts\activate
```

Далее установите все нужные зависимости из файла `requirements.txt`

```
pip install -r requirements.txt
```
