# Создаем новый проект

Когда вы создали виртуальное окружение и скачали нашу библиотеку, вы сразу же можете приступать к работе!

Создайте первый файл, например `main.py` и импортируйте туда библиотеку. Данный файл будет точкой входа в ваш проект. Создайте в нем экземпляр окна, в котром будет открывать ваша игра

```python
import alien

root = alien.Window() # Создаем экземпляр окна
root.size((900, 600)) # Здаем размеры окна
root.title('First game') # Задаем заголовок окна

root.loop() # Запускаем основной цыкл игры
```

Запустите файл командой

```
python main.py
```

У вас откроется ваше первое окно!

> [!WARNING]
> Запуская проект данной командой, вы запускаете его в тестовом режиме. Для полноценной работы соберите ваш проект. Подробнее читайте в документации - Финальная сборка
